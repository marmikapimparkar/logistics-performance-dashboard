## 🚀 Project Overview
This project transforms raw, inconsistent shipment data into actionable logistics insights.  
Using Microsoft Excel, I cleaned, structured, and analyzed 300+ shipments to uncover key business trends and performance metrics.

---

## 🔧 Key Highlights
- **Data Cleaning & Standardization**: Applied TRIM, CLEAN, and duplicate removal for consistency.
- **Custom Metrics**: Built formulas for Profit, Profit Margin, and Revenue per KM.
- **Performance Tracking**: Applied conditional logic to flag delayed shipments and payment status.
- **Data Integration**: Linked multiple datasets using XLOOKUP for seamless analysis.
- **Interactive Dashboard**: Designed Pivot Tables and charts for dynamic exploration.

---

## 📊 Insights Delivered
- Identified **top-performing regions and product categories**.
- Analyzed **monthly revenue and profit trends**.
- Detected **high-profit shipments vs. delayed shipments**.
- Delivered **KPI metrics** for logistics efficiency.
